# Contradiction-Detector
The AI Contradiction Detector is a lightweight and efficient Natural Language Processing (NLP) application designed to analyze the relationship between textual inputs. It helps identify whether two sentences support each other, contradict each other, or are unrelated.  
# Contradiction Detector

This project detects contradictions between two sentences using Machine Learning.

## Features
- NLP-based contradiction detection
- Streamlit UI
- Highlight contradictory words

## Tech Stack
- Python
- Scikit-learn
- Streamlit
